# Introduction

This repository is the source for my blog

# Getting Started


```bash
# install dependencies
npm install
```

```bash
# run eleventy (generate static site)
npx @11ty/eleventy
```

```bash
# build and host on a local development server
npx @11ty/eleventy --serve
```
